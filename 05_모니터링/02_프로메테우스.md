### 프로메테우스 사용법
```
    프로메테우스 환경 구성
        mkdir -p d:\data\prometheus001 d:\data\prometheus001\data d:\data\prometheus001\conf
        mkdir -p d:\data\prometheus002 d:\data\prometheus002\data d:\data\prometheus002\conf
        mkdir -p d:\data\prometheus003 d:\data\prometheus003\data d:\data\prometheus003\conf
        chmod -r 777  /data/prometheus00*

    프로메테우스 yml ==> /data/prometheus003/conf  로 복사
        
    프로메테우스 컨테이너 생성
        docker run -it --name prom001 -h prom001 -p 9090:9090 \
        --net mybridge --net-alias=prom001 \
        -v d:\data\prometheus001\data:/data \
        -v d:\data\prometheus001\conf:/etc/prometheus \
        -d prom/prometheus-linux-amd64

    프로메테우스 설정
        

    프로메테우스 실행 확인
        http://{docker_host_ip}:9090/graph



```
