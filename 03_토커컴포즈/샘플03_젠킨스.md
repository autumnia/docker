### docker-compose.yml  파일 작성 후 저장
```
version: "3"

services:
  jenkins:
    image: jenkins:latest
    container_name: jenkins

    hostname: jenkins
    ports:
      - "8080:8080"

    environment:
      -u root

    volumes:
      - /jenkins:/var/jenkins_home    # linux
      - d:\data\jenkins_data          # windows

    networks:
      - mybridge

```

### 실행 및 접속
```
  docker logs jenkins  로 패스워드 확인

  접속 URL
    http://{{hostIP}}:8080/  
      아이디: root  패스워드: 로그로 확인한 패스워드 복사 
```
