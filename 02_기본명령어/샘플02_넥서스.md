### 넥서스 생성 샘플
```
    넥서스 실행 방법
    docker run -it --name lm-nexus3 -h nexus3 -p 8081:8081 \
    -v d:/data/nexus-data:/nexus-dat              <= 윈도우
    -v /srv/docker-data/nexus-data:/nexus-data    <= 리눅스
    -d nexus3

    접속 URL
    http://{{hostIP}}:8080/  :8081/  
        아이디: admin  패스워드: admin

```
