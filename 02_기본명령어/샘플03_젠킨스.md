### 젠킨스 생성 샘플
```
    넥서스 실행 방법
    docker run -it --name jenkins -h jenins -p 8080:8080 \
    -v /jenkins:/var/jenkins_home  \        <= 리누스
    -v /d:/data/jenkins_data \
    -u root  \
    -d jenkins/jenkins:lts

    docker logs jenkins  로 패스워드 확인

    접속 URL
    http://{{hostIP}}:8080/  
        아이디: root  패스워드: 로그로 확인한 패스워드 복사 
```
